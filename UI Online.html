UI Online

To get started select an option below.

File a New Claim

Select File New Claim to complete the application. No additional UI Online registration is required to file a claim.
The File a New Claim features will be available during the times listed below. (All times are Pacific time)

Day of the Week Available Time
Sunday 5:00 a.m. - 8:30 p.m.
Monday 4 a.m. - 10 p.m.
Tuesday - Friday 2 a.m. - 10 p.m.
Saturday 2 a.m. - 8 p.m.

Register or Manage Existing Claim

Select Register or Manage if you would like to:
  • Register for UI Online with your existing claim.
  • Manage your existing UI Online account.
